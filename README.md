# Automation Tests

Acest proiect este un proiect de testare automată JavaScript cu Playwright.

## Comenzi

- `npm test` - rulează testele Playwright
- `npm run test:headed` - rulează testele cu browser vizibil
- `npm run test:report` - deschide raportul HTML Playwright

## Structură

- `tests/` - conține testele Playwright
- `playwright.config.js` - configurație Playwright
- `.gitignore` - fișiere ignorate de Git
